# Platform Directory Structure (SPECIFIED)

This tree was created deliberately as the initial executable platform monorepo for BonAcqui-LLC/cognitive-basin-platform.

Directories under packages/, python/, and apps/ are **SPECIFIED** (reserved and intentionally empty or lightly seeded). They are not claimed as IMPLEMENTED.

.gitkeep files are present only to reserve the namespace and contain explicit "SPECIFIED, not yet IMPLEMENTED" markers.

Evidence imports (canonical sources with provenance) go under evidence/legacy/.

New implementation code goes in the appropriate package with clear lineage comments.

See docs/ for architecture, ADR, and specifications.

This structure supports the canonical flow: PERCEPT → ATAL → RIGOR → CIRCUIT → GUARD → SERA and the understanding/memory layers.
