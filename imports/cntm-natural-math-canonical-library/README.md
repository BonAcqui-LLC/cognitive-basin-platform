# CNTM / Natural Math Canonical Library

Local canonical-source library for Natural Math, CNTM, Morphological Coding, Fractalish, MCVA / AMCVA / HOLD, Cognitive Basin, and SymLan-adjacent materials.

This pass preserves source artifacts, groups them by project lane, records hashes and provenance, and avoids forcing unresolved source conflicts into a false canonical state.
