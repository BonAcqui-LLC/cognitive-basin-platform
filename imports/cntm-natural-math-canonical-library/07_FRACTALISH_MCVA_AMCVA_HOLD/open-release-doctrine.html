Open Release Doctrine

Public first, but not careless.

This release is public-first. The aim is to establish a public record, make the ideas usable, invite criticism and improvement, and reduce the chance that the framework disappears behind obscurity or gatekeeping.

The core position is straightforward:

  • the core vocabulary is open,
  • the core tools are open,
  • the core atlas method is open,
  • the core documentation is open.

The core framework is open. If this work helps you, support it, cite it, contribute back, sponsor development, or hire us to apply it.

This release is not framed as patent pending, commercial lock-in, proprietary atlas access, a finished product, or a universal theory.

Open release does not mean free labor forever. Sustainability can come from voluntary support, sponsored development, implementation help, training, hosted tools, custom morphology reports, domain-specific analysis, and donations.

Do not publish private prompts, unpublished legal strategy, private medical or financial material, tokens, keys, secrets, or unlicensed third-party datasets.
