# Fractal Math Primer

Canonical mathematical fractals are useful reference anchors. They are not the whole project.

Fractalish uses canonical families such as:

- Mandelbrot-like boundary complexity,
- Koch-like iterative edge growth,
- Sierpinski-like recursive subdivision,
- Hilbert-like space filling,
- dragon-curve-like folded paths,
- branching tree anchors.

These become the alphabet. Physical morphology becomes the language.

Fractal dimension is one baseline descriptor. It estimates how occupied boxes scale as box size changes. It is helpful for scale-bounded complexity, but it cannot identify morphology by itself. Two images may share similar fractal dimension while differing sharply in topology, symmetry, periodicity, branch survival, wake geometry, or artifact load.

Fractalish therefore combines fractal dimension with topology, skeleton, lacunarity, anisotropy, curvature, symmetry, periodicity, branch structure, and trace geometry.
