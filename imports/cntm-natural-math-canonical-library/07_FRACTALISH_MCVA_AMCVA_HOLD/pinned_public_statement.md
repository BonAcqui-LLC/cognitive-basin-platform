# Pinned Public Statement

Use this as a pinned GitHub repo note, pinned site statement, or pinned social post.

We are publicly releasing the core Fractalish / MCVA framework.

The short version:

- MCVA is the readout layer for morphology-bearing process evidence.
- AMCVA is the guardrail atlas for absent, hidden, misleading, or non-diagnostic morphology.
- HOLD is the unresolved evidentiary state.
- Natural Math is the forward or generative local-process layer.
- SERA measures cost, waste, retries, false commitments, HOLD rate, and valid output.

This is not a medical diagnostic system, not a disaster predictor, not a universal theory, and not a finished product.

It is an open research framework for reading process from form, preserving uncertainty, and comparing before claiming.

The core framework is open. If this work helps you, support it, cite it, contribute back, sponsor development, or hire us to apply it.
