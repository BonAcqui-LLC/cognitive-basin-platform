## What This Project Is

This is an open research framework for reading process from form.

It tries to answer a narrow but useful question:

When does observable morphology preserve enough structure to support comparison, descriptor extraction, and process-memory hypotheses, and when should the system refuse to over-read?

## If You Want To Contribute

Good places to help first:

- improve wording and clarity,
- suggest stronger non-claims,
- add manifest structures,
- propose public-safe example domains,
- add or test simple descriptors,
- improve roads/public-works demos,
- identify AMCVA boundary cases,
- improve documentation for new readers.

## If You Only Read One Sentence

Read form carefully. Preserve uncertainty. Compare before claiming.
