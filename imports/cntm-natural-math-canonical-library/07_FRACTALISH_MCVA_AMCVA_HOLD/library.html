Reference Library

A public registry of traces, not just pretty pictures.

The library is where Fractalish stores canonical anchors, physical morphology examples, negative controls, and HOLD cases with trace files, metrics, and routing records attached.

Canonical anchors

Synthetic branching, irregular boundary, polygonal, crack, and noise examples provide a stable alphabet for the workbench before large public datasets are layered in.

Physical morphology families

Cracks, dendrites, vascular or transport networks, channels, porous fields, surface textures, wakes, and competing geometries should appear with the same evidence package structure.

Proof-positive / proof-negative / HOLD

The library is not only a celebration wall. It should preserve proof-negative and HOLD examples so the public record also shows where the framework refused or deferred a claim.

Current public state

The starter set focuses on synthetic anchors and generated sample records while broader public-license workflows are still being stabilized.
